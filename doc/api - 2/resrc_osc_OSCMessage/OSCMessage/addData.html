List<int> addData (String data)

Convert data to the required format for OSC message.

Implementation

List<int> addData(String data) {
  List<int> bytes = utf8.encode(data).toList();
  bytes.add(0);
  while (bytes.length % 4 != 0) {
    bytes.add(0);
  }
  return bytes;
}
